Introduction

This application demonstrates the basic functions of the SatTrackr-board. It is for test and presentation purpose.

procedure

This application runs the following setup procedure:

  • blinking LED once
  • opening Serial connection
  • sending welcome message
  • blinking LED twice
  • creating servo-object
  • sending serial message
  • blinking LED 3-times

Usage

The application reads in serial input and set servo motor to received position input string has to be terminated with a semicolon "," and hast to be in range between 0 and 180

Compilation Info

This software it can be compiled for arduino nano and the exported hex-file can be uploaded to the SatTrackr Board rev A
